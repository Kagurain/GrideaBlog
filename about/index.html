🤣 错误大赏

当您发现本网站有很多的框框框框时，那是 Emoji 无法正常显示，如果您认为影响了您的阅读体验，您可以尝试以下解决方法

1.更换浏览器：经查证，大部分浏览器都存在特定系统有些可以显示 Emoji （手机端安卓 5.0 和 iOS12 以上一般可以显示）
测试平台:
Windows10 X64 Chrome80 可以显示
Win7 X86 360极速浏览器(Chrome80内核)无法显示
Win10教育版 X64 原生Edge（非Chromium内核）可以显示
Win7 X86 搜狗浏览器 一台可显一台不可显
iPhone6 iOS12.43 夸克 可以显示
均为全彩Emoji

2.安装 Segoe UI Emoji
下载方式需自行百度，唯一不足的一点就是安装之后只有灰白Emoji，并且没有Emoji全适配
